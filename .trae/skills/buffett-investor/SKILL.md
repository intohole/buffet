## 5+1层分析框架

```
Layer 0: 前瞻与预判 (Forward-Looking) → 产业趋势/政策/技术/供需预判
Layer 1: 风险与纪律 (All Masters)     → 资金保全与情绪控制
Layer 2: 价值与护城河 (Buffett)       → 商业质量与内在价值 + 护城河耐久性
Layer 3: 成长分类 (Lynch)             → 股票类型识别 + 渗透率/替代率定位
Layer 4: 趋势与时机 (Livermore)       → 入场/出场时机 + 领先信号确认
Layer 5: 宏观与反射性 (Soros)         → 市场环境与泡沫检测 + 范式转换预判
```

## 热点→选股漏斗（核心工作流）

### Phase 1: 热点信息采集

**必须执行4批次并行WebSearch**（每批≥3个搜索，总计≥12次）。

**日期规则**: 搜索关键词中的日期必须使用系统当前日期。季度根据月份推算（1-3月=Q1，4-6月=Q2，7-9月=Q3，10-12月=Q4）。

```
批次1-市场面:
  "A股今日热点板块 涨停 {今日日期}"
  "主力资金流向 机构加仓 {今日日期}"
  "北向资金 今日流入 板块 {今日日期}"
  "两市成交额 量能变化 {今日日期}"

批次2-A股特色数据（必须执行）:
  "A股 涨停板 连板梯队 {今日日期}"
  "龙虎榜 机构席位 游资席位 {今日日期}"
  "涨停数量 跌停数量 封板率 {今日日期}"
  "换手率排名 量比排名 {今日日期}"

批次3-政策面+业绩面:
  "政策利好 产业政策 最新 {今日日期}"
  "业绩超预期 {当前季度}报 {当前年份}"
  "行业景气度 景气上行 {当前年份}"
  "净利润增长 业绩爆发 {当前年份}"

批次4-事件面:
  "突发新闻 财经 今日 {今日日期}"
  "全球市场 美股 美联储 影响"
  "大宗商品 原油 铜 黄金 价格"
```

**对每个候选热点，执行5项纵深搜索**:
```
1. "[热点名称] 龙头股 核心标的 {当前年份}"
2. "[热点名称] 订单 业绩 验证 {当前季度}报"
3. "[热点名称] 产业链 上中下游 受益股"
4. "[热点名称] 风险 争议 利空"
5. "[热点名称] 券商研报 目标价 评级 {当前年份}"
```

**反馈闭环**: 若纵深搜索发现业绩无法验证或存在重大风险，必须降低该热点评分或直接淘汰。

### Phase 2: 热点质量评估（4维12分制）

| 维度 | 高质量(3分) | 中等(2分) | 低质量(1分) |
|------|-------------|-----------|-------------|
| 政策支持 | 国家级政策，多年周期 | 部委/地方政策 | 无政策支持 |
| 资金流向 | 主力净流入持续3天+，金额>50亿 | 1-2天流入，金额<50亿 | 无明确流入或净流出 |
| 业绩验证 | 真实业绩增长确认 | 预期增长，尚未兑现 | 无业绩支撑，纯概念 |
| 板块广度 | 多股涨停(>5只)，梯队完整 | 1-2只龙头涨停 | 仅单只上涨 |

- ≥10分 = ✅可持续主线 → 重点关注，可配置15-20%仓位
- 6-9分 = ⚠️次级题材 → 适度参与，轻仓8-12%
- <6分 = ❌一日游 → 回避

### Phase 3: 垂直领域确认

| 检查项 | 通过标准 | 未通过处理 |
|--------|----------|-----------|
| 有中军股(大盘龙头稳健上涨) | 机构信心确认 | 降低仓位预期 |
| 板块指数突破20日均线 | 趋势确认 | 等待突破 |
| 资金连续3天净流入 | 资金确认 | 继续观察 |
| 有政策催化 | 政策确认 | 降低持续性预期 |

- 4项全通过=高确定性→15-20%仓位 | 3项=中等→8-12% | ≤2项=仅观察

### Phase 4: 上市公司筛选

**第一层: 龙头/中军/补涨分类**

| 分类 | 特征 | 仓位上限 |
|------|------|---------|
| 龙头 | 板块内首个涨停/连板数最高/封板坚决/基本面有催化 | ≤15% |
| 中军 | 大盘龙头/ROE>15%/稳健上涨/代表机构信心 | ≤20% |
| 补涨 | 同板块未涨/基本面不错但滞后 | ≤5% |

**A股龙头战法增强检查**（每个候选龙头必须执行）:

| 指标 | 通过标准 |
|------|---------|
| 龙虎榜 | 机构净买入 或 机构+游资同买 |
| 连板质量 | 早盘封板+封单>5%+换手5-15% |
| 筹码集中度 | 连续2季度股东户数减少>10% |
| 量比 | 当日量比2.5-5(明显放量) |
| 题材收入占比 | >5%(非蹭热点) |

**第二层: 5层框架快速筛选**

| 层级 | 通过标准 | 未通过处理 |
|------|----------|-----------|
| Layer 0 | 产业趋势可预判，非纯概念 | 淘汰 |
| Layer 1 | 能解释业务，非FOMO | 淘汰 |
| Layer 2 | 护城河≥5分，财务≥5/7 | 淘汰 |
| Layer 3 | 正确分类+PEG合理+渗透率定位 | 调整策略 |
| Layer 4 | 非派发期+领先信号确认 | 等待 |
| Layer 5 | 非系统性风险+范式未转换 | 降低仓位 |

**第三层: 割裂度筛选**

1. 板块背离: 板块回调时个股不跌 → 强主力
2. 大盘背离: 大盘创新低个股不跟 → 独立强势
3. 量价背离: 价涨量增价跌量缩 → 健康吸筹

3项全通过=强割裂 | 2项=中等 | ≤1项=弱(谨慎)

### Phase 5: 股票深度分析

对通过筛选的标的，执行7角度纵深搜索:
```
1. "[股票名] [代码] 财报 ROE 毛利率 净利润 {当前年份}"
2. "[股票名] 市占率 护城河 竞争优势"
3. "[股票名] 管理层 增持 分红 回购"
4. "[股票名] 券商研报 目标价 评级 {当前年份}"
5. "[股票名] 风险 利空 争议"
6. "[股票名] 技术分析 均线 MACD 趋势"
7. "[行业] 行业趋势 格局 增速 {当前年份}"
```

**A股红旗检查**（每个标的必须执行，严禁跳过）:

| 序号 | 检查项 | 危险信号 |
|------|--------|---------|
| 1 | 存贷双高 | 存款和贷款同时高企→财务造假风险 |
| 2 | 关联交易占比 | >30%→利益输送风险 |
| 3 | 商誉/净资产比 | >50%→减值风险 |
| 4 | 应收账款增速 | 远超营收增速→回款风险 |
| 5 | 经营现金流 | 连续3年负值→资金链风险 |
| 6 | 大股东质押率 | >50%→强平风险 |
| 7 | 审计意见 | 非标→财务可信度低 |
| 8 | ST/退市风险 | ST/*ST→禁止推荐 |
| 9 | 近期解禁 | 1周内大额解禁→回避 |
| 10 | 题材收入占比 | <1%→蹭热点，禁止参与 |
| 11 | 龙虎榜信号 | 机构卖出+游资买入=出货 |
| 12 | 尾盘首板 | 14:30后封板=信心不足 |
| 13 | 孤狼涨停 | 板块仅1只涨停=一日游 |
| 14 | 5板以上空间板 | 禁止推荐 |

**经验沉淀格式**:

每条经验必须包含：
```
### [日期] [经验标题]
- 情境：什么市场环境下遇到的
- 观察：发现了什么现象
- 验证：是否经过≥2次交叉验证
- 规则提炼：[可复用原则]
- 置信度：高/中/低（随验证次数提升）
- 关联经验：[链接到相关经验]
```

**经验锤炼规则**：
- 每次分析后，用 Grep 搜索 `investment-experience/` 中已有经验
- 若本次观察与已有经验一致，提升该经验置信度（低→中→高）
- 若本次观察与已有经验矛盾，降低置信度并标注"待验证"
- 置信度=高的经验可直接写入决策规则
- 置信度=低的经验仅作参考，不写入决策规则
- 前瞻性判断必须记录待验证假设和验证期限
