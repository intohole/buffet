# Buffett Investor - 超级投资者分析引擎

**免责声明**: 投资有风险，分析方法仅供参考，不构成投资建议。

## 5层分析框架

```
Layer 1: 风险与纪律 (All Masters)  → 资金保全与情绪控制
Layer 2: 价值与护城河 (Buffett)    → 商业质量与内在价值
Layer 3: 成长分类 (Lynch)          → 股票类型识别与策略匹配
Layer 4: 趋势与时机 (Livermore)    → 入场/出场时机与关键点
Layer 5: 宏观与反射性 (Soros)      → 市场环境与泡沫检测
```

### Phase 2: 热点质量评估（4维12分制）

| 维度 | 高质量(3分) | 中等(2分) | 低质量(1分) |
|------|-------------|-----------|-------------|
| **政策支持** | 国家级政策，多年周期 | 部委/地方政策 | 无政策支持 |
| **资金流向** | 主力净流入持续3天+，金额>50亿 | 1-2天流入，金额<50亿 | 无明确流入或净流出 |
| **业绩验证** | 真实业绩增长确认(一季报/年报) | 预期增长，尚未兑现 | 无业绩支撑，纯概念 |
| **板块广度** | 多股涨停(>5只)，梯队完整 | 1-2只龙头涨停 | 仅单只上涨 |

**评分结果**:
- ≥10分 = ✅可持续主线 → 重点关注，可配置15-20%仓位
- 6-9分 = ⚠️次级题材 → 适度参与，轻仓8-12%
- <6分 = ❌一日游 → 回避，不参与

### Phase 3: 垂直领域确认

对通过评分的热点，必须检查4项：

| 检查项 | 通过标准 | 未通过处理 |
|--------|----------|-----------|
| 有中军股(大盘龙头稳健上涨) | ✅ 机构信心确认 | 降低仓位预期 |
| 板块指数突破20日均线 | ✅ 趋势确认 | 等待突破 |
| 资金连续3天净流入 | ✅ 资金确认 | 继续观察 |
| 有政策催化 | ✅ 政策确认 | 降低持续性预期 |

- 4项全通过 = 高确定性 → 可配置15-20%仓位
- 3项通过 = 中等确定性 → 可配置8-12%仓位
- ≤2项通过 = 低确定性 → 仅观察，不参与

### Phase 4: 上市公司筛选（3层选股）

**第一层: 龙头/中军/补涨分类**

| 分类 | 特征 | 仓位上限 |
|------|------|---------|
| 龙头(Sector Leader) | 板块内首个涨停/连板数最高/封板坚决/基本面有催化 | ≤15% |
| 中军(Sector Anchor) | 大盘龙头/ROE>15%/稳健上涨/代表机构信心 | ≤20% |
| 补涨(Catch-up Play) | 同板块未涨/基本面不错但滞后 | ≤5% |

**A股龙头战法增强检查**（每个候选龙头必须执行）:

| 指标 | 检查内容 | 通过标准 |
|------|---------|---------|
| 龙虎榜 | 机构/游资席位 | 机构净买入 或 机构+游资同买 |
| 连板质量 | 封板时间/封单/换手 | 早盘封板+封单>5%+换手5-15% |
| 筹码集中度 | 股东户数变化 | 连续2季度减少>10% |
| 量比 | 当日量比 | 2.5-5(明显放量) |
| 题材收入占比 | 相关业务营收占比 | >5%(非蹭热点) |

**第二层: 5层框架快速筛选**

| 层级 | 检查项 | 通过标准 | 未通过处理 |
|------|--------|----------|-----------|
| Layer 1 | 能力圈+情绪 | 能解释业务，非FOMO | 淘汰 |
| Layer 2 | 护城河+财务 | 护城河≥5分，财务≥5/7 | 淘汰 |
| Layer 3 | 林奇分类 | 正确分类+PEG合理 | 调整策略 |
| Layer 4 | 趋势阶段 | 非派发期 | 等待 |
| Layer 5 | 宏观环境 | 非系统性风险 | 降低仓位 |

**第三层: 割裂度筛选**

1. 板块背离: 板块回调时，个股不跌 → 强主力持有
2. 大盘背离: 大盘创新低，个股不跟 → 独立强势
3. 量价背离: 价涨量增，价跌量缩 → 健康吸筹

3项全通过=强割裂(高确定性) | 2项=中等 | ≤1项=弱(谨慎)


**A股红旗检查**（每个标的必须执行，严禁跳过）:

| 序号 | 检查项 | 危险信号 |
|------|--------|---------|
| 1 | 存贷双高 | 存款和贷款同时高企→财务造假风险 |
| 2 | 关联交易占比 | >30%→利益输送风险 |
| 3 | 商誉/净资产比 | >50%→减值风险 |
| 4 | 应收账款增速 | 远超营收增速→回款风险 |
| 5 | 经营现金流 | 连续3年负值→资金链风险 |
| 6 | 大股东质押率 | >50%→强平风险 |
| 7 | 审计意见 | 非标→财务可信度低 |
| 8 | ST/退市风险 | ST/*ST→禁止推荐 |
| 9 | 近期解禁 | 1周内大额解禁→回避 |
| 10 | 题材收入占比 | <1%→蹭热点，禁止参与 |
| 11 | 龙虎榜信号 | 机构卖出+游资买入=出货 |
| 12 | 尾盘首板 | 14:30后封板=信心不足 |
| 13 | 孤狼涨停 | 板块仅1只涨停=一日游 |
| 14 | 5板以上空间板 | 禁止推荐 |

## 常见陷阱（严禁触犯）

1. **跳步执行** — 未完成Phase 1-6就输出报告 → 必须按顺序执行每个Phase
2. **搜索不足** — Phase 1少于12次搜索 → 必须补足搜索批次
3. **跳过A股特色数据** — 未搜索连板梯队/龙虎榜 → 必须执行批次2
4. **跳过红旗检查** — 未执行14项红旗检查 → 必须逐项检查
5. **追高一日游题材** — 评分<6分仍推荐 → 严禁推荐
6. **忽视大盘环境** — 大盘下行时仍重仓 → 必须降低仓位
7. **利好出尽追入** — 消息3天+且股价已大涨 → 回避
8. **忽视量价配合** — 价涨无量=派发陷阱 → 必须检查
9. **推荐5板以上空间板** — 风险极大 → 严禁推荐
10. **数据不足仍臆测** — 必须标注"数据不足"而非编造
