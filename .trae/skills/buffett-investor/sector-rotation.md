# Sector Rotation - 板块轮动与市场洞察

## 市场环境评估协议

### Step 1: 宏观数据采集

执行4批次并行WebSearch：

**批次1 - 市场面**:
```
- "上证指数 创业板指 科创50 走势 [当前日期]"
- "两市成交额 量能变化 [当前日期]"
- "融资融券余额 [当前日期]"
```

**批次2 - 政策面**:
```
- "央行 货币政策 降准降息 [当前日期]"
- "证监会 监管政策 最新"
- "财政政策 专项债 赤字率"
```

**批次3 - 全球面**:
```
- "美股 纳斯达克 标普500 走势"
- "美联储 利率决议 影响"
- "地缘政治 中美关系 贸易"
```

**批次4 - 估值面**:
```
- "A股 估值 PE分位 PB分位 [当前日期]"
- "沪深300 中证500 估值水平"
- "股债性价比 风险溢价"
```

### Step 2: 市场环境评分 (7因素)

| 因素 | 看多(+1) | 中性(0) | 看空(-1) |
|------|---------|---------|---------|
| **利率环境** | 降息周期/低利率 | 持平 | 加息周期/高利率 |
| **信用环境** | M2增速>10%，社融扩张 | 持平 | 信用收紧，社融收缩 |
| **市场情绪** | 融资余额合理，北向流入 | 持平 | 融资余额飙升，北向流出 |
| **政策方向** | 明确支持资本市场 | 持平 | 监管收紧 |
| **估值水平** | PE<历史30%分位 | 持平 | PE>历史70%分位 |
| **成交量** | 放量上涨 | 持平 | 缩量下跌 |
| **趋势方向** | 指数>MA60 | 持平 | 指数<MA60 |

**评分结果**:
- **≥4分 = 多头环境 (BULL)** → 积极配置，仓位≤60%
- **1-3分 = 中性环境 (NEUTRAL)** → 均衡配置，仓位≤40%
- **≤0分 = 空头环境 (BEAR)** → 防御配置，仓位≤20%

### Step 3: 板块轮动分析

#### 经济周期→板块映射

| 经济周期 | 特征 | 超配板块 | 标配板块 | 低配板块 |
|---------|------|---------|---------|---------|
| **复苏期** | GDP↑，利率低 | 科技成长、可选消费 | 金融、工业 | 必选消费、公用事业 |
| **扩张期** | GDP↑，利率↑ | 金融、工业、原材料 | 必选消费 | 科技成长(估值压力) |
| **放缓期** | GDP↓，利率高 | 必选消费、医药 | 公用事业、高股息 | 周期品、可选消费 |
| **收缩期** | GDP↓，利率↓ | 高股息、红利资产 | 债券、黄金 | 周期品、成长股 |

#### 板块强度排名方法

执行WebSearch获取板块数据：
```
- "板块涨幅排名 今日 [当前日期]"
- "板块资金流向 主力净流入 [当前日期]"
- "板块成交额 排名 [当前日期]"
```

**强度评分** = 涨幅权重40% + 资金流向权重40% + 成交额权重20%

### Step 4: 反射性检查

执行泡沫检测清单(7项)和繁荣-萧条周期定位。

| 检查项 | 状态 |
|--------|------|
| 价格远超内在价值 | ☐ |
| "这次不一样"叙事 | ☐ |
| 散户大量涌入 | ☐ |
| 杠杆率快速上升 | ☐ |
| 聪明钱离场 | ☐ |
| 媒体狂热报道 | ☐ |
| 政策开始收紧 | ☐ |

**泡沫预警**: ≥4项 → 全面减仓

### Step 5: 哑铃策略配置

根据市场环境评分和反射性检查结果，确定哑铃策略配置：

| 环境 | 成长端 | 防御端 | 现金 |
|------|--------|--------|------|
| 多头(≥4分) | 50-60% | 20-30% | 10-20% |
| 中性(1-3分) | 30-40% | 30-40% | 20-30% |
| 空头(≤0分) | 10-20% | 40-50% | 30-40% |

### Step 6: 生成市场洞察报告

使用 [report-templates.md](report-templates.md) 中的"市场洞察报告"模板输出。

## 板块轮动交易规则

### 入场规则

| 规则 | 条件 | 优先级 |
|------|------|--------|
| 政策催化+资金流入确认 | 政策出台+北向/主力流入3天+ | 最高 |
| 突破20日均线+放量 | 板块指数突破MA20，量>1.5倍均量 | 高 |
| 优先中军 | 选择板块内大盘龙头(ROE>15%) | 高 |
| 首次回调入场 | 突破后首次回调至MA20 | 中 |

### 出场规则

| 规则 | 条件 | 紧急程度 |
|------|------|----------|
| 跌破20日均线 | 板块指数跌破MA20 | 🟡 减仓50% |
| 跌破60日均线 | 板块指数跌破MA60 | 🔴 清仓 |
| 资金流向反转 | 北向/主力连续3天净流出 | 🟡 减仓 |
| 政策转向 | 监管收紧/政策退出 | 🔴 立即离场 |

### 仓位配置

| 板块类型 | 多头环境 | 中性环境 | 空头环境 |
|---------|---------|---------|---------|
| 主导板块 | 15-20% | 8-12% | 3-5% |
| 次级板块 | 8-12% | 5-8% | 0-3% |
| 投机板块 | 3-5% | 0-3% | 0% |
| 现金 | ≥10% | ≥20% | ≥40% |

## 聪明钱跟踪 (Smart Money Tracking)

### 北向资金信号

| 信号 | 判断 | 操作 |
|------|------|------|
| 连续3天+净流入某板块 | 看多 | 关注该板块龙头 |
| 连续3天+净流出某板块 | 看空 | 减仓或回避 |
| 单日大幅流入(>5倍均值) | 潜在催化 | 立即研究 |
| 沪股通流入+深股通流出 | 风格切换 | 从成长切向价值 |

### 融资融券信号

| 信号 | 判断 | 操作 |
|------|------|------|
| 融资余额快速上升 | 狂热预警 | 谨慎，考虑减仓 |
| 融资余额快速下降 | 潜在底部 | 关注反弹机会 |
| 融资余额>2.6万亿 | 极度亢奋 | 全面减仓 |

### 机构持仓信号

| 信号 | 判断 | 操作 |
|------|------|------|
| 公募基金加仓比例>15% | 强烈看多 | 跟踪关注 |
| 社保基金新进 | 长期看好 | 中期持有 |
| 大股东增持 | 内部人信心 | 正面信号 |

## 哑铃策略 (Dumbbell Strategy)

根据市场环境调整配置：

### 多头环境 (评分≥4)

```
成长端 50-60%:  AI算力、半导体、新能源
防御端 20-30%:  高股息、消费龙头
现金   10-20%:  等待回调加仓
```

### 中性环境 (评分1-3)

```
成长端 30-40%:  精选龙头，控制仓位
防御端 30-40%:  高股息、医药、公用事业
现金   20-30%:  保留安全边际
```

### 空头环境 (评分≤0)

```
成长端 10-20%:  仅保留最优质龙头
防御端 40-50%:  高股息、黄金、债券
现金   30-40%:  等待底部信号
```

## 反射性检查 (Reflexivity Check)

在市场洞察分析中，必须执行反射性检查：

### 泡沫检测清单 (≥4项→泡沫预警)

1. ☐ 价格远超内在价值(PE>历史均值2倍+)
2. ☐ "这次不一样"叙事盛行
3. ☐ 散户大量涌入(新开户数激增)
4. ☐ 杠杆率快速上升(融资余额飙升)
5. ☐ 聪明钱开始离场(大股东/机构减持)
6. ☐ 媒体狂热报道(财经杂志封面=见顶信号)
7. ☐ 政策开始收紧(监管信号)

### 繁荣-萧条周期定位

| 阶段 | 特征 | 操作 |
|------|------|------|
| 认知偏差 | 新趋势出现，少人注意 | 研究，建观察名单 |
| 繁荣 | 资金涌入，价格上涨 | 信念投资 |
| 狂热 | "这次不一样" | 减仓 |
| 危机 | 聪明钱离场 | 激进出局 |
| 崩溃 | 恐慌性抛售 | 准备购物清单 |
