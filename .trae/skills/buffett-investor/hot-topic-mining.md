# Hot Topic Mining - 热点挖掘与题材分析

核心工作流: **热点挖掘 → 垂直领域确认 → 上市公司筛选 → 股票深度分析 → 汇总输出**

## Phase 1: 热点信息采集 (深度搜索)

### 执行并行WebSearch（至少4组，每组3-5个搜索）

**搜索批次1 - 市场面**:
```
- "A股今日热点板块 涨停 {今日日期}"
- "主力资金流向 机构加仓 {今日日期}"
- "北向资金 今日流入 板块 {今日日期}"
- "两市成交额 量能变化 {今日日期}"
```

**搜索批次1.5 - A股特色数据** (热点模式必做):
```
- "A股 涨停板 连板梯队 {今日日期}"
- "龙虎榜 机构席位 游资席位 {今日日期}"
- "涨停数量 跌停数量 封板率 {今日日期}"
- "换手率排名 量比排名 {今日日期}"
```

**搜索批次2 - 政策面**:
```
- "政策利好 产业政策 最新 {今日日期}"
- "政府工作报告 投资主线 十五五"
- "国务院/工信部/证监会 最新政策 行业 {今日日期}"
- "创业板/科创板 改革 政策"
```

**搜索批次3 - 业绩面**:
```
- "业绩超预期 一季报/年报 {当前季度} {当前年份}"
- "行业景气度 景气上行 {当前年份}"
- "机构调研 最受关注 {当前月份}"
- "净利润增长 业绩爆发 {当前年份}"
```

**搜索批次4 - 事件面**:
```
- "突发新闻 财经 今日 {今日日期}"
- "行业重大事件 催化剂"
- "全球市场 美股 美联储 影响"
- "大宗商品 原油 铜 黄金 价格"
```

### 深度搜索增强

对初步发现的热点，执行**纵深搜索**（每个候选热点5项搜索）：

```
对每个候选热点:
1. "[热点名称] 龙头股 核心标的 {当前年份}"
2. "[热点名称] 订单 业绩 验证 一季报"
3. "[热点名称] 产业链 上中下游 受益股"
4. "[热点名称] 风险 争议 利空"
5. "[热点名称] 券商研报 目标价 评级 {当前年份}"
```

**反馈闭环**: 若纵深搜索发现业绩无法验证或存在重大风险，则降低该热点评分或直接淘汰。

## Phase 2: 热点质量评估 (4维12分制)

| 维度 | 高质量(3分) | 中等(2分) | 低质量(1分) |
|------|-------------|-----------|-------------|
| **政策支持** | 国家级政策，多年周期(如十五五) | 部委/地方政策 | 无政策支持 |
| **资金流向** | 主力净流入持续3天+，金额>50亿 | 1-2天流入，金额<50亿 | 无明确流入或净流出 |
| **业绩验证** | 真实业绩增长确认(一季报/年报) | 预期增长，尚未兑现 | 无业绩支撑，纯概念 |
| **板块广度** | 多股涨停(>5只)，梯队完整(龙头+中军+补涨) | 1-2只龙头涨停 | 仅单只上涨，无跟风 |

**评分结果**: 
- ≥10分 = 可持续主线 (SUSTAINABLE) → 重点关注，可重仓
- 6-9分 = 次级题材 (SECONDARY) → 适度参与，轻仓
- <6分 = 一日游 (AVOID) → 回避，不参与

## Phase 3: 垂直领域确认

### 板块定位

对通过评分的热点，确认其所属垂直领域：

1. **产业链定位**: 该热点属于哪个产业链的上/中/下游？
2. **经济周期匹配**: 当前经济周期阶段，该板块是否为主导？
3. **政策周期匹配**: 当前政策周期是否支持该板块？

参考 [sector-rotation.md](sector-rotation.md) 中的板块轮动框架。

### 垂直领域筛选标准

| 检查项 | 通过标准 | 未通过处理 |
|--------|----------|-----------|
| 有中军股(大盘龙头稳健上涨) | ✅ 机构信心确认 | 降低仓位预期 |
| 板块指数突破20日均线 | ✅ 趋势确认 | 等待突破 |
| 资金连续3天净流入 | ✅ 资金确认 | 继续观察 |
| 有政策催化 | ✅ 政策确认 | 降低持续性预期 |

**4项全通过 = 高确定性垂直领域** → 可配置15-20%仓位
**3项通过 = 中等确定性** → 可配置8-12%仓位
**≤2项通过 = 低确定性** → 仅观察，不参与

## Phase 4: 上市公司筛选 (3层选股)

### 第一层: 龙头/中军/补涨分类

```
Priority 1: 龙头股 (Sector Leader)
  - 板块内首个涨停
  - 连板数最高
  - 最大市值+强势走势
  - 基本面有实质催化(订单/业绩)
  - 龙虎榜有机构/知名游资买入
  - 封板坚决(秒封/早盘封板)
  - 适合: 短线+中线

Priority 2: 中军股 (Sector Anchor)
  - 大盘龙头，基本面强(ROE>15%，护城河宽)
  - 稳健上涨(非涨停但持续)
  - 代表机构资金信心
  - 最适合中期持有(1-3月)

Priority 3: 补涨股 (Catch-up Play)
  - 同板块，尚未上涨
  - 基本面不错但价格滞后
  - 低风险但低回报
  - 仅在龙头确认板块强度后入场
  - 仓位≤5%
```

**A股龙头战法增强** (参考 a-stock-specifics.md):

对每个候选龙头，额外检查:

| A股指标 | 检查内容 | 通过标准 |
|---------|---------|---------|
| 龙虎榜 | 机构/游资席位 | 机构净买入 或 机构+游资同买 |
| 连板质量 | 封板时间/封单/换手 | 早盘封板+封单>5%+换手5-15% |
| 筹码集中度 | 股东户数变化 | 连续2季度减少>10% |
| 量比 | 当日量比 | 2.5-5(明显放量) |
| 题材收入占比 | 相关业务营收占比 | >5%(非蹭热点) |

### 第二层: 5层框架快速筛选

对每个候选标的，执行5层快速扫描：

| 层级 | 检查项 | 通过标准 | 未通过处理 |
|------|--------|----------|-----------|
| Layer 1 | 能力圈+情绪 | 能解释业务，非FOMO | 淘汰 |
| Layer 2 | 护城河+财务 | 护城河≥5分，财务≥5/7 | 淘汰 |
| Layer 3 | 林奇分类 | 正确分类+PEG合理 | 调整策略 |
| Layer 4 | 趋势阶段 | 非派发期 | 等待 |
| Layer 5 | 宏观环境 | 非系统性风险 | 降低仓位 |

详细方法论参考: [master-methods.md](master-methods.md)

### 第三层: 割裂度筛选 (Divergence Filter)

**核心原则**: "顺境中的涨停不一定强，逆境中逆势涨停才值得重视。"

1. **板块背离**: 板块回调时，个股不跌 → 强主力持有
2. **大盘背离**: 大盘创新低，个股不跟 → 独立强势
3. **量价背离**: 价涨量增，价跌量缩 → 健康吸筹

**割裂度评分**: 3项全通过=强割裂(高确定性) | 2项=中等 | ≤1项=弱(谨慎)

## Phase 5: 股票深度分析

对通过筛选的标的，执行完整5层深度分析。

参考 [deep-analysis.md](deep-analysis.md) 中的详细流程。

### 深度搜索模式

对每个重点标的，执行7角度纵深搜索：

```
1. 财务数据: "[股票名] [代码] 财报 ROE 毛利率 净利润 {当前年份}"
2. 竞争地位: "[股票名] 市占率 护城河 竞争优势"
3. 管理质量: "[股票名] 管理层 增持 分红 回购"
4. 券商观点: "[股票名] 券商研报 目标价 评级 {当前年份}"
5. 风险因素: "[股票名] 风险 利空 争议"
6. 技术分析: "[股票名] 技术分析 均线 MACD 趋势"
7. 行业背景: "[行业] 行业趋势 格局 增速 {当前年份}"
```

A股特有检查参考: [a-stock-specifics.md](a-stock-specifics.md)

## 热点题材类型与应对

| 类型 | 驱动力 | 持续性 | 最佳入场 | 退出信号 |
|------|--------|--------|----------|----------|
| 政策驱动型 | 国家政策 | 2-4周(国家级) / 1-2周(部委) | Day 1-2 | 政策落地后3天 |
| 事件驱动型 | 业绩/产品/并购 | 1-5天 | 事件确认前 | 利好出尽 |
| 技术驱动型 | AI/半导体/新能源 | 数周-数月 | 回调确认后 | 业绩不达预期 |
| 资金驱动型 | 北向/融资 | 不定 | 流入确认3天+ | 流出3天+ |

## 避坑指南

1. **追高一日游题材** — 评分<6分，不追
2. **忽视大盘环境** — 大盘下行时，减少所有热点仓位
3. **只看龙头不看中军** — 无中军=无机构信心=大概率一日游
4. **利好出尽追入** — 消息3天+且股价已大涨，回避
5. **忽视量价配合** — 价涨无量=派发陷阱
6. **不做纵深搜索** — 仅看表面信息，不验证业绩和风险
7. **忽视A股特有风险** — 存贷双高、关联交易等红旗未检查
8. **忽视增速放缓** — 从25%降至15%，即使仍高于市场，股价也可能暴跌
9. **追5板以上空间板** — 风险极大，禁止参与
10. **忽视龙虎榜信号** — 机构卖出+游资买入=出货信号，不参与
11. **忽视解禁日历** — 大额解禁前后1周回避
12. **题材收入占比<1%** — 蹭热点，非真正受益
13. **尾盘首板** — 14:30后封板=资金信心不足
14. **孤狼涨停** — 板块仅1只涨停=大概率一日游
