# Workflow Checklists

热点→选股漏斗的检查清单已在SKILL.md中定义。本文件仅包含其他模式的检查清单。

## 前瞻性选股 工作流

详细步骤: [forward-looking.md](forward-looking.md)

```
Step 1: 产业趋势扫描
  ☐ 3批次并行WebSearch(产业趋势/政策前瞻/前瞻信号)
  ☐ 识别候选趋势(渗透率/替代率/技术迭代)

Step 2: 趋势定位
  ☐ 渗透率S曲线定位(概念/导入/加速/成长/成熟)
  ☐ 国产替代率定位(起步/加速/爆发/深化/成熟)
  ☐ Gartner技术迭代定位(萌芽/膨胀/破裂/爬升/成熟)
  ☐ 综合定位矩阵判定

Step 3: 信号验证
  ☐ 领先信号扫描(≥2个)
  ☐ 确认信号验证(≥1个)
  ☐ 滞后信号排除(无危险组合)
  ☐ 预期差评估(市场共识vs实际趋势)

Step 4: 标的筛选
  ☐ 产业链传导定位(上游/中游/下游)
  ☐ 5+1层框架快速筛选
  ☐ A股红旗检查(14项必查)
  ☐ 割裂度筛选

Step 5: 输出报告
  ☐ 按报告模板输出(含前瞻性分析章节)
  ☐ 前瞻判断标注待验证假设和验证期限
```

## 选股筛选 工作流

详细步骤: [deep-analysis.md](deep-analysis.md) Mode A

```
Step 1: 确定筛选条件
  ☐ 风险偏好(保守/均衡/进取)
  ☐ 投资期限(短期/中期/长期)
  ☐ 板块偏好
  ☐ 资金规模

Step 2: 多源数据采集
  ☐ 执行4批次并行WebSearch(价值/成长/股息/板块)
  ☐ 收集10-20只候选标的

Step 3: 5+1层过滤
  ☐ Layer 0 前瞻与预判(产业趋势定位+信号验证)
  ☐ Layer 1 风险与纪律(能力圈+情绪检查)
  ☐ Layer 2 门控(财务≥5/7，护城河≥5分)
  ☐ Layer 3 分类(林奇6型+PEG+渗透率定位)
  ☐ Layer 4 时机(买入信号≥4/6+领先信号确认)
  ☐ Layer 5 宏观(环境有利+范式转换检查)

Step 4: 排名与选择
  ☐ 综合评分(护城河30%+财务25%+安全边际20%+时机15%+宏观10%)
  ☐ 输出前5-10名
```

## 买卖时机 工作流

详细步骤: [deep-analysis.md](deep-analysis.md) Mode B

```
Step 1: 质量门控确认
  ☐ 标的通过Layer 2质量门控
  ☐ 若未通过→告知用户"时机分析对劣质股无意义"

Step 2: 深度技术分析
  ☐ 关键均线(5/10/20/60/120MA)
  ☐ MACD/RSI状态
  ☐ 成交量趋势
  ☐ A股特色数据(龙虎榜/换手率/量比/筹码/解禁)

Step 3: 入场/出场点判定
  ☐ 识别入场信号(关键点突破/均线支撑/MACD金叉/RSI超卖/放量反转)
  ☐ 识别出场信号(护城河侵蚀/派发形态/MACD死叉/RSI超买/止损触发)
  ☐ 若买入信号<4/6→判定WAIT

Step 4: 生成时机报告
  ☐ 判定: BUY/WAIT/SELL/AVOID
  ☐ 入场区间+仓位+止损+目标价
```

## 持仓诊断 工作流

详细步骤: [deep-analysis.md](deep-analysis.md) Mode C

```
Step 1: 收集持仓信息
  ☐ 读取 user-data/ 下的持仓/风格/操作日志
  ☐ 提醒用户补充待补充信息

Step 2: 逐只5+1层分析
  ☐ 对每只持仓执行完整5+1层分析
  ☐ A股红旗检查(14项必查)
  ☐ 前瞻信号验证(领先指标是否出现)

Step 3: 组合层面评估
  ☐ 集中度检查(单股≤20%，单板块≤30%)
  ☐ 现金储备(≥10%)
  ☐ 质量漂移(任一持仓未通过Layer 2→立即卖出)
  ☐ 相关性(过度集中于同一宏观风险→分散化)

Step 4: 生成持仓报告
  ☐ 每只持仓评级(卖出/减仓/持有/加仓)
  ☐ 调仓计划
```

## 市场洞察 工作流

详细步骤: [sector-rotation.md](sector-rotation.md)

```
Step 1: 宏观数据采集
  ☐ 执行4批次并行WebSearch(市场面/政策面/全球面/估值面)

Step 2: 市场环境评分
  ☐ 7因素评分(利率/信用/情绪/政策/估值/成交量/趋势)
  ☐ 总分判定: ≥4多头 / 1-3中性 / ≤0空头

Step 3: 板块轮动分析
  ☐ 经济周期定位
  ☐ 板块强度排名
  ☐ 超配/标配/低配建议

Step 4: 反射性检查
  ☐ 泡沫检测清单(7项)
  ☐ 若泡沫预警≥4项→建议全面减仓

Step 5: 哑铃策略配置
  ☐ 根据环境评分确定成长端/防御端/现金比例
```

## 复盘迭代 工作流

详细步骤: [review-loop.md](review-loop.md)

```
Step 1: 执行复盘(每次分析后)
  ☐ 检查数据完整性
  ☐ 检查流程完整性
  ☐ 检查判断一致性
  ☐ 检查输出规范性

Step 2: 生成复盘文档
  ☐ 文件名: docs/review-YYYYMMDD-HHmm.md
  ☐ 分析概要+核心判断依据+前瞻性判断+风险预警+待验证假设
  ☐ 经验沉淀引用(链接到 investment-experience/ 中的具体条目)

Step 3: 归因复盘(3-5个交易日后)
  ☐ 判断准确性: 评级与实际走势是否一致？
  ☐ 时机精确度: 入场价是否接近最优？
  ☐ 归因分类: 能力 vs 运气

Step 4: 策略复盘(每月1次)
  ☐ 胜率统计
  ☐ 方法论有效性
  ☐ 策略迭代
  ☐ 经验锤炼(提升/降低置信度)
  ☐ 前瞻准确性(前瞻判断验证通过率)
```
